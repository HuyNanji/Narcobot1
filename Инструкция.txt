1. Скачать питон последней версии с офф.сайта
2. При установке нажать галочку на Path 
3. Как установиться питон, откройте командную строку (cmd)
4. Перекиньте файлы на рабочий стол
5. Открыть файл config.py ( открывать лучше через notepad+++ )
6. Внести свой айди телеграмма @getmyid_bot
7. Внести токен с созданного бота через БотФактор @BotFather
7.1 Как создать бота в нем можно найти инструкцию в инетернете.
8. Запустить cmd
9. Вбить команду pip install pytelegrambotapi 
10. Вбить команду cd desktop - переносит нас на рабочий стол
11. Вбить команду python main.py - Запуск бота. Готово


Если вышла ошибка по поводу библиотеки можно попробовать вписать pip install telethon
